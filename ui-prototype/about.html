关于 HelloTime Pro

这是一个多技术栈对比学习项目 —— 同一款时光胶囊 Web 应用， 由 10 个后端、5 个前端、5 个全栈框架各自独立实现一遍， 像一张可切换的技术雷达。前端打开浏览器开发者工具就能看见后端响应的具体栈。

ⓘ 项目处于 M0 (Design) 阶段 · 2026-04-19 · 所有 UI 都基于同一套 spec/ tokens 实现；切换后端只需调整 .env 里的 BACKEND 变量。

SQLite · 备选驱动

三条硬约束

  • API 同源：所有后端实现同一份 spec/api/openapi.yaml，用契约测试验证。
  • UI 同源：所有前端消费同一份 spec/styles/tokens.css，视觉对齐到像素级。
  • 数据同源：所有后端兼容同一份 spec/db/schema.sql，可通过反向代理热切换。
